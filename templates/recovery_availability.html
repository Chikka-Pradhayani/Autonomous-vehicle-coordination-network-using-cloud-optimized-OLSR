🔬 Technical Analysis

🔄 OLSR Recovery Mechanism

  • Periodic HELLO messages detect link failures within 2-3 intervals
  • TC (Topology Control) messages propagate updates via MPRs
  • Shortest Path First calculation provides optimal routes instantly
  • No route discovery delay - routes always available

📊 DSDV Recovery Mechanism

  • Full routing table broadcasts required for updates
  • Sequence number propagation delays route convergence
  • Incremental updates may miss critical changes
  • Higher overhead during network instability
